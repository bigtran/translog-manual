# translog 简介

## translog 是什么？
简单来说，是一个基于webman的博客系统，用户通过简单的安装配置，就可以拥有自己的站点。使用markdown语言撰写文章，并发布成网页。

## translog 能做什么？
目前为止，我用它来：
- 做个人博客站点
- 做团队博客站点
- 做在线接口文档站点
- 做内部访问的会议记录站点（加密访问）
- 做...

## translog 的特点
- markdown 撰写文章
- yaml 描述站点信息、页面信息
- html + php 来设计页面
- webman 驱动做 server + app
- nginx 方向代理做网址访问
